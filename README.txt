*Note: make sure python is installed on the PC
when installing python make sure you tick add to PATH
After intalling python open cmd and type
pip install virtualenv


***********Steps to run the server**************
1. open to the first (hhhBlog which contains virt and hhhBlog)
2. In the folder navigate to the navigation input field clear everything and type cmd
3. when cmd opens activate the virtual environment by typing the command below
4. virt\Scripts\activate.bat and press enter
5. After (virt) shows up, type cd hhhBlog in the cmd once again and press enter
6. finally, type python manage.py runserver and press enter
7. below is what you will see
